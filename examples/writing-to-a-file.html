Writing to a File

#include <fstream>
#include <iostream>

using namespace std;

int main() {
  ofstream outfile;
  // Open the file in append mode
  outfile.open("filename.txt", ios_base::app);
  if (!outfile) {
    cerr << "Unable to open file for writing";
    return 1;
  }
  // Write to the file
  for (int i = 1; i <= 20; i++) {
    outfile << i * i << endl;
  }
  // Close the file
  outfile.close();
  return 0;
}

Core concepts
...

  • #include <fstream> required in order to interact with files
  • ofstream outfile stands for output file stream and it’s used to create files and write information to files.
  • outfile.open("filename.txt", ios_base::app); The outfile.open method in C++ is used to open a file for writing data. The first parameter is the specific file to open and the ios_base::app flag opens the file in append mode.
  • cerr is an object in C++ that is used to output error messages. It is an instance of the ostream class and is defined in the iostream header file. cerr is unbuffered which allows it to display the error message immediately without storing it for later.
  • outfile << This lets you write to files by using the bitwise shift operator to stream data out to the file. In this case, it will append the data to the file due to the ios_base::app.
  • outfile.close(); closes the output file and is good practice for managing memory.
