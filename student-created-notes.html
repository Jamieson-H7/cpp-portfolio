Variables
...

Variables are used to store data in memory.
int x = 5;

Fundamental Data Types:
...
  • int: Integer data type, used to store whole numbers.
  • float: Single-precision floating-point numbers.
  • double: Double-precision floating-point numbers.
  • char: Character data type, used to store a single character.
  • bool: Boolean data type, used to store either true or false.
Other Data Types:
...
  • Array: A collection of elements of the same data type.
  • Pointer: A variable that stores the memory address of another variable.
  • Reference: A reference to an existing variable.
  • Function: A variable that stores a pointer to a function.
Modifiers:
...

Modify variables data types

  • const: Indicates that a variable's value cannot be changed after initialization.
  • volatile: Indicates that a variable can be changed at any time by external factors. Mainly used to ensure that the compiler doesn't optimize necessary components of the code.
  • mutable: Used in classes to specify that a member variable can be modified even in a const member function.

Functions
...

Functions group code together. A function is code that performs a specific task.
Example:

int add(int x, int y) {
    return x + y;
}

Here's a function “add” that takes two integer parameters and returns their sum.
Example:

int result = add(3, 4);
cout << result << endl; // 7
Void Functions
...

These functions do not return any value.

void printHello() {
	std::cout << "Hello, World!" << std::endl;
}
Functions with Return Values
...

These functions will return the value specified.

int add(int a, int b) {
    return a + b;
}
Inline Functions
...

Expands the function in place at the call site rather than being executed as a separate function.
Pros/Cons: reducing function call overhead, enabling compiler optimization, increasing code size, and causing thrashing

inline int square(int x) {
	return x * x;
}

Structures
...

Structures are created data types that allow grouping of different data types.
Example:

struct Person {
    string name;
    int age;
    string address;
};

In this the structure “Person” has three members of “name”, “age” and “address”.

You can create variables of this structure type

Person p1;
p1.name = "John";
p1.age = 30;
p1.address = "123 Main St.";

You can also access the members of the structure using the dot operator

cout << p1.name << endl;
cout << p1.age << endl;
cout << p1.address << endl;

Classes
...

In C++, classes are used to create objects that have properties and methods.
Example:
class Person {
public:
string name;
int age;
void say_hello() {
cout << "Hello, my name is " << name << " and I am " << age << " years old." << endl;
}
};

In this example, we have a class "Person" that has two properties of "name" and "age". We've also defined a method called "say_hello" that outputs a greeting to the console.

Objects
...

Person p1;
p1.name = "John";
p1.age = 30;

Method Calls
...

p1.say_hello();

This will output "Hello, my name is John and I am 30 years old." to the console.
